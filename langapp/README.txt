==============================================
🙏 BHASHA — Indian Language Learning App
==============================================

KAISE CHALAYEIN (How to Run):
--------------------------------
1. Python 3.8+ install hona chahiye

2. Dependencies install karo:
   pip install -r requirements.txt

3. App chalao:
   python app.py

4. Browser mein jaao:
   http://localhost:5000

FEATURES:
----------
✅ Login / Register system
✅ Progress save hoti hai (SQLite database)
✅ 4 Languages: Marathi, English, Marwadi, Sanskrit
✅ Flashcard-style lessons
✅ MCQ Quizzes with scoring
✅ XP points system
✅ Daily streak tracking
✅ Profile page with stats

DUSRON KO SHARE KARNA:
-----------------------
- Same WiFi pe ho toh: http://YOUR_IP:5000
- Public share ke liye "ngrok" use karo:
  1. ngrok download karo (ngrok.com)
  2. python app.py chalao
  3. ngrok http 5000 chalao
  4. Jo URL milega woh share karo!
